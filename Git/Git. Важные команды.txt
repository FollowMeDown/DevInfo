
Восстановление потерянных коммитов
----------------------------------
1. git reflog - выводит последние действия и коммиты
2. находим и копируем нужный коммит 'abc'
3. git checkout 'abc'


Поиск истории удаленных файлов
------------------------------
http://stackoverflow.com/questions/7203515/git-how-to-search-for-a-deleted-file-in-the-project-commit-history

If you do not know the exact path you may use
git log --all --full-history -- **/thefile.*

If you know the path the file was at, you can do this:
git log --all --full-history -- <path-to-file>


Удаление ТЕГОВ на сервере
-------------------------
https://stackoverflow.com/questions/5480258/how-to-delete-a-git-remote-tag
https://stackoverflow.com/questions/9705534/how-do-i-delete-a-git-branch-with-tortoisegit
git push --delete origin tagname


Удаление ВЕТОК на сервере и локально
------------------------------------
https://stackoverflow.com/questions/2003505/how-do-i-delete-a-git-branch-both-locally-and-remotely?noredirect=1&lq=1
git push origin --delete <branch_name>
git branch -d <branch_name>


Удаление текущих неиспользуемых веток, если они были удалены на сервере
-----------------------------------------------------------------------
https://stackoverflow.com/questions/9705534/how-do-i-delete-a-git-branch-with-tortoisegit
git remote update origin --prune
или
git fetch --prune <--all>


Возврат файлов последнего коммита, чтобы их можно было отредактировать и заново закоммитить
-------------------------------------------------------------------------------------------
git reset --soft HEAD~1
или
git reset --soft <branch_name>
